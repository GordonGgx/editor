# FxMarkdownEditor #

使用JavaFx编写的开源Markdown编辑器，代码用于学习和分享使用。

![image](image.png)
### Feature ###

* 编辑器语法高亮
* 查找和替换
* HTML时事预览
* 文件树结构目录层次
* 编辑，预览，编辑/预览切换
* 支持Windows,Linux
